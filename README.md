# 🛡️ BMC-ECONOMICS

**BMC-ECONOMICS** is a high-performance, open-source academic portal engineered to provide students with a streamlined, searchable interface for mastering **Economics**.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg?style=for-the-badge)
![License](https://img.shields.io/badge/license-MIT-green.svg?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg?style=for-the-badge)

## 🚀 Key Features

* **Global Economics Search**: A real-time filtering engine allowing students to isolate specific subjects instantly.
* **YouTube API Integration**: High-fidelity video playback that respects creator rights and native embeds.
* **Ambient UI Design**: A glassmorphic interface featuring a blurred "Ambient" background for a premium aesthetic.
* **Interactive Legal Layer**: Built-in modal system for License agreements and Anti-Piracy compliance.
* **Low-Bandwidth Optimized**: Engineered to be lightweight for students with limited internet connectivity.

## 📖 How to Use

1.  **Exploration**: Enter the portal and browse featured economics subjects.
2.  **Discovery**: Use the **Search Bar** to pinpoint specific topics (e.g., Microeconomics, Macroeconomics) instantly.
3.  **Legal**: Review the **License** modal to understand usage rights and academic compliance.
